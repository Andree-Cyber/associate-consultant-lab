# Client Scenario

A small business with 25 employees currently uses separate local computer accounts across employee workstations. This setup creates security and management challenges because user accounts, passwords, and access permissions are managed individually on each machine.

The business needs a centralized login system, stronger password rules, organized departments, shared folders, and basic access control.

## Business Goals

- Improve identity and access management
- Simplify employee onboarding
- Reduce IT management time
- Enforce stronger password and lockout policies
- Organize users by department
- Provide secure shared folder access

## Proposed Solution

The proposed solution is to implement Active Directory Domain Services using Windows Server 2022. The environment includes a domain controller, organized Organizational Units, department-based users, security groups, shared folders, Group Policy settings, and a Windows 10 domain-joined workstation.
